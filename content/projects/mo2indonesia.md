MO2 Indonesia is a marketplace and event management platform built for the automotive industry, connecting merchants, racing communities, and event organizers in one ecosystem.

## Overview

A multi-functional platform designed to support automotive merchants and racing event organizers.

The system enables:

* Merchants and automotive stores to showcase and manage their products
* Event organizers to publish and manage racing events
* Racers to register for competitions
* Category selection based on race class
* Motorcycle number selection
* Online payment for event registration

The platform centralizes marketplace functionality and racing event management into a single integrated system.

## Stack

* Laravel
* MySQL
* Payment Gateway Integration
* Role-Based Access Control
* Cloud Hosting
