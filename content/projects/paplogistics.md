PAP Logistics is an internal web application developed to streamline logistics operations, shipment calculations, and container management.

## Overview

A centralized logistics management system built to handle operational workflows such as shipment calculation, container assignment, delivery coordination, and driver performance tracking.

The system enables administrators to:

- Calculate item volume, cubic measurement, weight, and quantity (koli/motor)
- Assign shipments into specific containers
- Track container allocation and shipment status
- Manage pickup and delivery assignments
- Assign drivers for delivery routes
- Monitor and generate driver performance reports

Designed to improve operational efficiency, accuracy, and real-time tracking within the company.

## Stack

- Laravel
- Livewire
- MySQL
- TailwindCSS
- Role-Based Access Control

Built using a server-driven architecture with Livewire to enable dynamic interfaces without heavy frontend frameworks.
